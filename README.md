# Medication Safety Prediction for Pregnant Women

## Overview

This project uses Machine Learning and Explainable Artificial Intelligence (XAI) techniques to predict medication safety during pregnancy. The system classifies medications into four categories:

* Safe
* Caution
* Dangerous
* Unknown

The project combines structured pharmaceutical data, text analysis of side effects, and explainability methods to support safer healthcare decision-making.

---

Install the required Python packages before running the notebook:

```bash
pip install numpy pandas matplotlib seaborn scipy nltk scikit-learn imbalanced-learn shap lime
```

## Running the Project

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd <repository-folder>
```

3. Open Jupyter Notebook:

```bash
jupyter notebook
```

4. Run all notebook cells sequentially from top to bottom.

---

## Project Workflow

The notebook consists of the following stages:

1. Data Loading
2. Target Variable Mapping
3. Data Cleaning and Preprocessing
4. Exploratory Data Analysis (EDA)
5. NLP Preprocessing
6. TF-IDF Feature Extraction
7. Feature Engineering
8. Train/Test Split
9. SMOTE Oversampling
10. Model Training (Random Forest & SVM)
11. Model Evaluation
12. Explainable AI (SHAP & LIME)
13. Final Results Summary
